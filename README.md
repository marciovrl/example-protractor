# example-protractor

Example of using the protractor for automated e2e testing.

# Preconditions:

- Node installed;
- NPM installed;
- Chromedriver installed.

# Use this project as a template:

- Clone the project:

```
git clone https://github.com/marciovrl/example-protractor.git
```

- Install dependencies

```
npm install
```

- Run webdriver:

```
npm run wd-start
```

- Run the testes

```
npm run test
```
